# Spec: 账号定位驱动的热点内容生产引擎

## 目标 (Goals)
- 基于热点证据和账号定位生成高质量单篇图文。
- 支持“每次只生成一篇，不满意可重生”。
- 账号定位在 UI 可见、可修改、可回滚，保存后全局生效。
- 图片内容先生成占位规划，后续再接真实生图。

## 非目标 (Non-goals)
- 本期不接 nano banana 实际生图 API。
- 本期不依赖外部账号 API 才能运行。

## 约束 (Constraints)
- 模型通过兼容网关调用，生产建议开启 `LLM_STRICT_MODE=true`。
- 人工仅做“使用/重生”选择，不做强制审核门禁。
- 重生默认不设上限。

## 范围 (Scope)
### In Scope
- `AccountProfile` + `AccountProfileVersion` 数据模型。
- `/api/accounts/:id/profile` 与回滚接口。
- 单篇生成、重生、图片占位规划接口。
- 草稿详情返回质量报告、内容包、占位规划、重生追踪。
- 内容生产页内联账号定位编辑 + 独立账号设置页。

### Out of Scope
- 实际图片生成、媒体上传与素材管理。

## 验收标准 (Acceptance)
- 账号定位可在 UI 编辑并保存，历史版本可回滚。
- 单篇生成结果包含完整结构和质量评分。
- 重生结果与上一稿保持高差异。
- 草稿详情可查看质量报告和图片占位。

## 风险与回滚 (Risk / Rollback)
- 风险：模型权重较高导致阶段性波动。
- 风险：无限重生带来成本增长。
- 回滚：按原子提交回退；账号定位可回滚历史版本。
